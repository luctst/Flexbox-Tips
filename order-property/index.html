Logout

  • you can change the order of your elements by using order.
  • By default all our elements are order:0.
  • The elements are visually placed in terms of the number in order.
  • So in our example the logout div which is in the third position in our html file has order:-1, because all our elements have order:0 by default we can give -1 to our element who will always move him to the first place.
  • If many elements have the same number in order the elements will be placed in terms of the order of the html file.
